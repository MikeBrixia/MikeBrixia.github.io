What is a Behavior Tree?

The Blackboard is the brain of the tree, it's an asset that can be created from the Unity Editor UI and assigned to a behavior tree. This asset can store around access game data at runtime and it's used by the tree to make it's own decisions.
Blackboard properties can be used to check conditions in decorators or their values can be updated using service nodes.

User guide

1. To create a new blackboard asset go to Assets->Create->AI->Blackboard

2. Once created, you can start adding new keys to the blackboard by pressing the "+" icon, for each key you will be asked to choose the key name and type. N.B. always keep in mind that each blackboard key name should be unique, duplicate names inside the same asset will generate conflicts.

3. The blackboard support a limited number of types, for more info about supported types check the BlackboardSupportedTypes API page

For more info about how to set and access blackboard keys at runtime take a look at the Blackboard API
